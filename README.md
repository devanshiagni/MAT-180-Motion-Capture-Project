# MAT-180-Motion-Capture-Project

This project is a Googel Colab notebook that takes a BVH file of motion capture data from the Carnegie Mellon Graphics Lab Motion Capture Database. The data I have chosen is the data of a modern dance. 

## How to Run
1. Download the file: "05_01.bvh"
2. Open the notebook in Google Colab
3. Run all cells
4. When asked to choose file, upload "05_01.bvh"

## Results/Outputs
Various Graphs are outputed. There is a graph output for each body part removal and full body. There are PCA graphs and Scree Plots. There is also a graph for the Cumulative Variance Across Experiments at the end of the Google Colab notebook.
